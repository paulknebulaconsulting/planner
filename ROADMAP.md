# Weekly Planner - Future Roadmap

This project is a starting point for a powerful, user-friendly weekly planner. Here are potential future features and improvements:

## Phase 2: User Experience Enhancements
- [ ] **Task Resizing**: Direct drag-to-resize on the bottom edge of task cards to adjust duration.
- [ ] **Color Coding**: Allow users to assign colors to tasks or categories for better visual organization.
- [ ] **Keyboard Shortcuts**: Common actions like `N` for new task, `Esc` to close modals, and `Ctrl+Z` to undo moves.
- [ ] **Dark Mode**: Support for system-wide dark mode preferences.
- [ ] **Tooltips & Onboarding**: Guided tour for new users to explain the drag-and-drop mechanics.

## Phase 3: Advanced Functionality
- [ ] **Recurring Tasks**: Setup tasks that repeat daily, weekly (e.g., every Tuesday), or on specific days.
- [ ] **Conflict Indicators**: Visual warnings when tasks overlap or when a day is over-scheduled.
- [ ] **Multi-Week Support**: Navigate between different weeks, copying common tasks from one week to the next.
- [ ] **Analytics**: Simple dashboard showing planned vs. actual time (if logging is added) or distribution of task categories.

## Phase 4: Integration & Synchronization
- [ ] **Google/Outlook Calendar Sync**: Two-way synchronization with external calendars.
- [ ] **Export to PDF/Image**: High-quality export of the weekly plan for printing.
- [ ] **User Accounts**: Move beyond `localStorage` to a backend (like Supabase or Firebase) for cross-device syncing.
- [ ] **Mobile App**: A dedicated mobile interface or a React Native version for on-the-go planning.

## Phase 5: Collaborative Features
- [ ] **Shared Plans**: Allow multiple users to view and edit a shared weekly plan (e.g., for families or small teams).
- [ ] **Comments & Notifications**: Discuss tasks within the app and receive reminders.
